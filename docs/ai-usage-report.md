# AI Tools Usage Report

## Claude (Anthropic)

**Used for:** Code generation 

Claude was used to generate all code in this project from scratch, including:

- **HTML** — full page structure (navbar, hero, projects, contact, footer sections)
- **CSS** — layout using CSS Grid and Flexbox, dark/light theme system with CSS custom properties, responsive design for mobile and tablet, and visual polish (hover effects, transitions, card styling)
- **JavaScript** — dark/light mode toggle with localStorage persistence, smooth scrolling, hamburger menu for mobile, contact form validation, and scroll-reveal animations via IntersectionObserver

---

## ChatGPT (OpenAI)

**Used for:** Image generation

ChatGPT was used to generate the visual assets used in the portfolio, including the profile illustration displayed in the hero section and the project preview images for each project card.
